Overview



A project of this magnitude requires data from multiple sources. We will go over the sources that were utilized to build this page, as well as the transformations applied to digest the information in a visual manner.

The data used in our visualizations can be found under the Data used buttons in each category. The link to the source of the data can be found using the source buttons under their respective category.

Global Plastic Production

Sourced from the Plastic Pollution article by Hannah Ritchie which is hosted on ourworldindata.org we used this data in conjunction with the Extrapolated change in plastic fate to 2050 data which is also found on ourworldindata.org.

Plastic Waste by Country

Similar to the Global Plastic Production data, the Plastic waste by country was taken from ourworldindata.org. The contents of the file were then injected into a GEOJSON file. This file was then used to create a cloropleth map on the Source page.

Plastic Waste by Sector

Similar to the Global Plastic Production data, the Plastic waste by Sector was taken from ourworldindata.org and converted into a json. The json file was then used to create the Plastic Waste by Sector donut chart.

Plastic Garbage Fate

As mentioned above, this data was combined with the Global Plastic Production data to create The Fate of Plastics Produced since 1950 figure on the source page.

The combined data set can be found in the Data Used button below.

Impact Studies Data

At this point we have a bit of a theme going and to further add to this, the Impact Studies were also scraped from ourworldindata (they're a really amazing resource that we highly recommend checking out!). This data was pretty self-explanatory so we didn't do any further transformation to it.

The same chart was then created with the additional ability to filter on the Effect page.

Surface Ocean Plastic

Similar to the some of the visualization sources above, this data was pulled from the Plastic Pollution article on ourworldindata. Specifically the Surface Plastic Mass by Ocean Basin chart. the only modification that was made to this was, the global total was removed.

The data was then converted to a JSON file where it was then used to create a leaflet map on the Effect page.

Global Cleanup Event Data

This data was pulled from the amazing website coastalcleanupdata.org. Check them out if you have time, they're doing some amazing work and their website has a fantastic interactive map for visualizing cleanup data.

We broke the data up into two pieces, the first was to split up and summarize the data based on each category. This data was then used to create a d3-funnel chart to visualize all of the different items collected.

The second piece of the data was to re-summarize the data based on GPS coordinates. This, along with leaflet and mapbox, were used to create an interactive map to visualize the cleanups that occured in 2019. Again, we highly encourage you to go checkout cleanupdata.org, they're doing some awesome stuff to help everyone around the globe.

The visualizations can be found on the Resolutions page.

*NOTE* The cleanup Map data is quite large as it has records predating 2019. This means it will take some time to load.
